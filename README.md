## ReuniTe: Social Media App

### Quick Start

Preview Link: https://reunite-social-media.vercel.app/

### Features Available

- Post Features
  - Add Post
  - Delete Post
  - Like Post (and Unlike Post)
  - Bookmark Post (and remove from Bookmark)
  - Add Comment on Post (Delete Comment)
- Profile Features
  - Edit Profile
  - Add Profile Picture
  - Add Bio
  - Add Website
- Follow Unfollow
  - Follow User
  - Unfollow User
  - visit user page
- Authentication
  - Login
  - Signup
  - Guest Login
- Fully Resposive

### Tech Stack

- **Frontend**: React, Redux Toolkit, React Router V6
- **Backend**: Mockbee
